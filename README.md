# App_Dev

Steps to make it:
- npx create-react-app image-labeling-app
- cd image-labeling-app
- npm install

Follow this dir structure:
image-labeling-app/
│── public/
│   ├── images/  (Store images here)
│   ├── labels.txt (This will store labels)
│── src/
│   ├── components/
│   │   ├── ImageLabeler.js
│   │   ├── ImageList.js
│   ├── App.js
│   ├── index.js
│── package.json
